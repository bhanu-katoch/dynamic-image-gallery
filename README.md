# 📸 Dynamic Image Gallery with Effects

An interactive web-based image gallery that allows users to dynamically add and remove images, along with applying smooth animations like fade, zoom, and transitions using jQuery.

---

## 🚀 Features

- 📂 Add images dynamically
- ❌ Remove images from gallery
- ✨ jQuery animations (fade, zoom, slide)
- 🖱️ Click-based interactive effects
- 📱 Responsive design
- ⚡ Real-time DOM updates

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- jQuery
